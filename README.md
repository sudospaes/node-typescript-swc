# Node.js TypeScript + SWC Template
A Node.js project template using TypeScript with SWC compiler and configured to ES module.

## Preparation for developing :
1 - Install packages.
```bash
npm i
```
2 - Make a build from project for the first time.
```bash
npm run build
```
3 - From now on.
```bash
npm run dev
```

## Run for production :
```bash
npm start
```
